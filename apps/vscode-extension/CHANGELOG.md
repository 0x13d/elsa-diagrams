# Changelog

## 0.4.0 - 2026-05-17

Editing readme and polishing the release pipeline.

## 0.1.0 — 2026-05-13

- Initial release.
- Mermaid preview to the side for Elsa Workflow JSON files (v2.x and v3+).
- Live updates as the JSON is edited.
- Direction toggle (TD / LR / BT / RL) in the preview toolbar.
- "Export to Markdown" command (and toolbar button) that writes a `.md` file
  with the rendered Mermaid in a fenced code block.
- Theme-aware Mermaid theming following the VS Code light/dark preference.
